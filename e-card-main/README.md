# Small Village & City Development Agency (SVCDA)

A comprehensive digital platform connecting rural and urban communities through one app — spanning Education, Grocery, Business, Health, Electronics, Agriculture, Services, Skill Development & Employment sectors.

## Tech Stack

- **Frontend**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS + shadcn/ui
- **State**: React Query + Context API
- **Routing**: React Router v6

## Getting Started

```bash
npm install
npm run dev
```

## Sectors

1. 🎓 Education — Schools, Colleges, Stationery, Skill Courses, Digital Learning
2. 🛒 Home Grocery — Local Stores, Online Ordering, Categories, Home Delivery
3. 💼 Business — Registration, Promotion, Startup Support, Networking
4. ❤️ Health — Doctor Appointments, Hospitals, Medicine, Health Awareness
5. 📱 Electronics — Repair Services, Buy Electronics
6. 🌾 Agriculture — Farming Tips, Fertilizers & Seeds, Machinery
7. 🔧 Services — Electricians, Plumbers, Carpenters, Painters & more
8. 📚 Skill Development — VCDA Mahila Shiksha Mission, Training, Certifications
9. 👥 Employment — Awareness, Jobs, Registration & Resume Upload
